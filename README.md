# 银行黑灰产智能监控与可视化分析平台

## 项目概述
本项目利用现代前端技术栈（Vue 3 + Vite + D3.js）开发，专门用于银行黑灰产行为的可视化监控与分析。系统通过图数据库和复杂网络分析算法，帮助金融机构快速识别和打击金融犯罪行为。

## 核心功能模块

### 1. 关系图谱可视化模块
- 使用 D3.js Force-Directed Graph 展示复杂网络
- 支持节点拖拽、缩放、聚焦等交互操作
- 智能布局算法，自动优化图谱展示效果

### 2. 动态过滤分析模块
- 金额阈值筛选：按交易金额范围过滤可疑节点
- 时间范围筛选：支持自定义时间区间分析
- 类型分类过滤：按行为类型分类展示

### 3. 关键链路追踪模块
- 自动识别并高亮显示关键资金流动路径
- 支持手动选择节点查看关联路径
- 路径权重分析，识别核心中转节点

### 4. 风险预警模块
- 实时数据更新与可视化
- 异常模式自动检测
- 风险等级颜色编码

## 使用方法
1. npm install（确保安装node.js）
2. npm run dev
